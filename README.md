# Merchant Admin Panel

This project implements the **Payments List page** for a merchant admin panel using **Next.js** and **Devias Kit Pro**.

## Tech Stack
- Next.js
- Devias Kit Pro (Next.js)
- Material UI

## Features
- Payments list with pagination
- Status-based filtering (All, Successful, Pending, Failed, Cancelled)
- OAuth2 client-credentials authentication (server-side)
- Secure API communication via Next.js API routes
- Devias dashboard layout and components

## Setup Instructions

1. Clone the repository
2. Copy `.env.example` to `.env.local`
3. Fill in the required environment variables
4. Install dependencies:
   npm install
